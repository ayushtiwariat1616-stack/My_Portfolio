# 🌐 **Portfolio Website – Ayush Tiwari**

A modern and aesthetic **personal portfolio website** built using **HTML, CSS, and JavaScript**.
It highlights my skills, projects, and contact details through a clean UI, smooth animations, and an interactive sidebar navigation system.

---

## 🚀 **Features**

* 🎨 Modern gradient UI & aesthetic design
* 📱 Fully responsive across all screen sizes
* 📂 Projects section with images
* 🎛️ Sidebar navigation with animation
* 👤 About section with skills
* 🧩 Micro-interactions on hover
* 📮 Contact section with form layout
* ⚡ Fast & lightweight — no framework required

---

## 🛠️ **Tech Stack**

* **HTML5**
* **CSS3**
* **JavaScript**
* Tools: *Figma, Arduino, ESP8266, Git*
